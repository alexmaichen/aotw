## Category of the Week : 🧊 Fresh File 📁
Earlier I alluded to how this is the aspect one starts the game with. How fast can you go in those circumstances? It's time to find out.
Speedrunners will tend to look for <:MercifulEnd:1051868903422300170> (the powerhouse boon), while someone looking for their first run 1 clear may prefer :FUNNYORANGEBOON: instead (best boon in Fresh File for ease of clearing). Outside of that, you know the drill, it's just a monodash sword run with unusual consistency-rules.

[Nitrostarch's WR](<https://www.speedrun.com/hades/runs/z1l36o9m>) is likely the best showing of what a near-perfect run through first run looks like.

## Category of the Week : :ice_cube: Fresh File :file_folder:
The premise of this cotw is simple: start a new savefile and Slay the S- uh, Father, as quickly as possible, regardless of if that is done first-run or not.
Several routes exist, but the fastest ones are [this](<https://scientific-paprika-b40.notion.site/Hades-Fresh-File-First-Run-Info-2d647029f5c34252acbd38ad5e54b221>) for sword, and [this](<https://www.speedrun.com/hades/guides/ha5fo>) for bow (except the current meta strategy for bow is to die in athenaroom on run 1, to have a second hammer three runs later, as well as the guaranteed c1 hammer).
There are to my knowledge no official written guides for the other weapons, though AotW writeups have contained a makeshift one before (feel free to check out).

Oh, we'll also be tracking your fastest Fresh File To Credits (FFTC) runs this week. A guide for that can be found [here](<https://docs.google.com/document/d/1IDDsaoZ_DSCk2TMg8UjDB4RLXc7IKvL9t-2GM9q2nx0/edit?tab=t.0>) and a one for the "No Unlocks" subcategory [here](<https://docs.google.com/document/d/1eqRSjZ1Rx6WNH2EXrpuxgRQs_KhMhDg0GCr_naTBIfQ/edit?tab=t.0>).
